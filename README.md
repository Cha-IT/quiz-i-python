# Quiz i Python

Dere skal lage en quiz i Python. Spørsmålene i quizen kan være om hva du vil. Quizen skal fungere sånn at ett og ett spørsmål skrives ut, med alternativer. Brukeren skal skrive inn hvilket alternativ de tror er riktig (tall eller bokstav). Dere velger selv om brukeren skal få svar med en gang om svaret er riktig eller galt. Antall riktige svar skal telles opp, og når quizen er ferdig skal brukeren få vite poengsummen sin, med en passende kommentar ut fra hvor mange poeng de har fått.

## Vanskelighetsgrad

Denne oppgaven kan gjøres så enkel eller vanskelig som du vil: 
Du kan for eksempel
* skrive spørsmålene etter hverandre, og sjekke hvert spørsmål etter at brukeren har gitt svar
* du kan lagre spørsmålene og svarene i en liste eller ordbok og hente dem ut derfra
* du kan lage en løkke som går gjennom hvert spørsmål
* du kan lage en funksjon som tar inn spørsmål og svar
* osv...
